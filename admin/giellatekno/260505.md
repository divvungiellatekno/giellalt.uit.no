# JÁV

Mange JÁV-artikler er like Sammallahti sine. Vi vil ikkje bruke artiklar som er identiske i dei to ordbøkene

Splitte opp:
    1. en-til-en ordpar (OK)
    2. Artikler JÁV har utvida (OK)
    3. Artikler som ikke er i Sammallahti (OK)
    4. Artikler som er helt like Sammallahti (Ikke OK)

1 og 3 er lett å finne, 2 og 4 kan være vanskeligere. Trond Ty jobbar vidare med dette.
