GNATS bug tracking system

  • To use GNATS, see http://localhost/cgi-bin/gnats/gnatsweb.pl
  • You should not need user names or accounts
  • Administrating gnats requires root rights
  • Gnats consists of gnats 4 beta, available with installation instructions from gnu.org, and the compatible gnatsweb 4 package from gnats/contrib/gnatsweb/
  • You can type locate gnats to find the files related to gnats. The files in cgi-bin belong to gnatsweb. The /usr/local/com directory contains the database named "default", which resides in gnatsdb/. /usr/local/etc/gnats contains global configuration files
  • Categories can be added as root like this.
