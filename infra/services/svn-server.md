# SVN Server

Everything related to the SVN Server we have, at `gtsvn.uit.no`. `gtsvn.uit.no` is
an alias for `gtsvn-02.uit.no` (we previously had `gtsvn-01.uit.no`).

The only service the svn server runs, is the svn server program.

## Browse

Browse the tree at <https://gtsvn.uit.no/langtech/>.

If the page does not appear, start the apache server by logging into gtsvn and running the command:

```
sudo service httpd start
```
