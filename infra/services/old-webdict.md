# Old webdict

The old webdict is still available at <https://gtweb.uit.no/old-webdict/>.
It runs on the [gtweb](../servers/gtweb.html) server.

The source tree is located at <https://github.com/giellatekno/webdict>.

When built, the old webdict is a completely static site, and is hosted directly
as such on the global nginx.

_The "new"_ [webdict](webdict.html) _is a rewrite of the old webdict, using the
same ideas, but with new code, and an updated look._
