# Oahpa (portal)

The oahpa portal is the site that runs on <https://oahpa.no/>. It contains links
to the various [Oahpa programs](oahpa-programs.html), as well as some learning
material, and other useful links.

It runs on the [gtoahpa](../servers/gtoahpa.html) server, where it builds from
svn sources.

Oahpa portal is a static site, built with
[Apache Forrest](../forrest/index.html), so as long as the main nginx web server
at gtoahpa is up, the site should be up. If it does not build, it's most likely
a bug in the source.
