# giellatekno.uit.no

[giellatekno.uit.no](https://giellatekno.uit.no) Giellatekno's website.

It runs on the [divvun](servers/divvun.html) server, where it builds from the
svn sources.

giellatekno.uit.no is a static site, built with
[Apache Forrest](../forrest/index.html), so as long as the main web server is
up, the site should be up. If it does not build, it's most likely a bug in the
source.
