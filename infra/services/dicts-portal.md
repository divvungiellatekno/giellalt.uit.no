# dicts.uit.no

[dicts.uit.no](https://dicts.uit.no) is a dictionary portal, linking to both
Divvun and Giellatekno dictionaries, but also others.

It runs on the [divvun](../servers/divvun.html) server, where it builds from the
svn sources.

dicts.uit.no is a static site, built with [Apache Forrest](../forrest/index.html),so as long as the main apache web server at divvun is up,
the site should be up. If it does not build, it's most likely a bug in the source.
